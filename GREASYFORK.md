## 功能特性

- 📦 **支持夸克网盘**：个人文件和分享链接
- 📦 **支持天翼云盘**：个人文件和分享链接
- 📁 **文件夹批量处理**：自动递归扫描文件夹
- 🎯 **自动生成秒传 JSON**：标准 123pan 格式
- 🔐 **支持加密分享**：自动处理密码保护的分享链接
- 🚀 **一键操作**：复制到剪贴板或下载 JSON 文件

## 使用方法

### 夸克网盘

#### 个人文件
1. 访问夸克网盘
2. 勾选文件或文件夹
3. 点击"生成JSON"按钮

#### 分享链接
1. 打开夸克分享链接
2. 首次使用需设置 Cookie（脚本会提示）
3. 点击"生成JSON"按钮

### 天翼云盘

#### 个人文件
1. 访问天翼云盘
2. 勾选文件或文件夹
3. 点击"生成JSON"按钮

#### 分享链接
1. 打开天翼分享链接
2. 输入提取码（如需要）
3. 勾选文件后点击"生成JSON"

## 配合使用

推荐配合 [123FastLink](https://github.com/Bao-qing/123FastLink) 使用：

1. 使用本脚本生成秒传 JSON
2. 使用 123FastLink 客户端导入
3. 快速转存到 123 云盘

**Q: 夸克分享链接提示 Cookie 失效？**
A: 点击错误提示中的"修改Cookie"按钮，按 F12 获取最新 Cookie 并粘贴

**Q: 如何获取夸克 Cookie？**
A: 打开开发者工具(F12) → Network → 刷新页面 → 找到任意请求 → 复制请求头中的完整 Cookie

**Q: 生成的 JSON 如何使用？**
A: 使用 123FastLink 客户端导入 JSON，即可秒传文件到 123 云盘
